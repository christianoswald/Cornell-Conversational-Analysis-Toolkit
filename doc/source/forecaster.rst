Forecaster
==========

Conversational Forecasting Transformers.

Refer to :doc:`CRAFT Model <craftmodel>` and :doc:`Cumulative Bag-of-Words <cumulativebow>` for the models that Forecaster can be loaded with.

Example usage: `CRAFT forecasting of conversational derailment <https://github.com/CornellNLP/Cornell-Conversational-Analysis-Toolkit/blob/master/convokit/forecaster/CRAFT/demos/craft_demo_new.ipynb>`_.

Example usage: `Forecasting of conversational derailment using a cumulative bag-of-words model <https://github.com/CornellNLP/Cornell-Conversational-Analysis-Toolkit/blob/master/convokit/forecaster/tests/cumulativeBoW_demo.ipynb>`_.
